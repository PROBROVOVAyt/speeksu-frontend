# Speeksu FrontEnd

speeksu-frontend - Это FrontEnd часть сайта Speek.su

## установка зависемостей

```bash
npm i
```

## запуск

```bash
npm run dev
```

## сборка

```bash
npm run build
```

## технологии

В данном проекте используются:

- Node.JS (версия: 20)
- Vite (версии: 6)
- React (версия: 18)
- React Router (версия: 7)
- TailwindCSS (версии: 3)
